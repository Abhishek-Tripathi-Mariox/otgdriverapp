# OTG Delivery Partner App

React Native CLI app for OTG delivery partners.

## Stack

- React Native CLI (0.85)
- NativeWind v4 (Tailwind CSS for RN)
- Zustand (state management)
- React Navigation (native stack)

## Setup

```bash
npm install
# iOS
cd ios && pod install && cd ..
npm run ios
# Android
npm run android
```
